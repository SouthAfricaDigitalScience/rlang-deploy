# rlang-deploy

Build, test and deploy scripts for the R language in CODE-RADE. 

# Versions

We build versions: 

  * 3.0.0

# Dependencies

The project is built depending on the following upstream CODE-RADE products : 

  * gcc (5.4.0)
  * openblas (0.2.15-gcc-5.4.0)
  * lapack (3.6.0-gcc-5.4.0)
  * jdk (8u66)
  * ncurses
  * readline
  * bzip2
  * xz
  * openssl (1.0.2j)
  * curl
  * pcre
  * zlib
  * icu (59_1-gcc-5.4.0)
  * jpeg (9b)
  * libpng (1.6.27)
